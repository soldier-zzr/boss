# boss_auto_greet

BOSS 直聘自动化招聘插件（本仓库用于代码备份）。

## 核心能力
- 推荐候选人页自动探测与任务启动/停止
- 先查看完整简历，再做本地规则+AI双层筛选
- 安全测试模式与真实模式切换
- 随机节奏/限频/阶段休息，模拟人工操作
- 自动聊天流程（先要简历，再按配置要微信）
- 飞书入库（Webhook / Bitable）
- 结构化日志与运行指标
